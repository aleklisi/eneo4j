Overview

eneo4j - Erlang client to neo4j graph database.

This is an application to conveniently communicate with neo4j using its HTTP API.

Installation

To your rebar.config add:

{deps, [
    ...
    {eneo4j, ".*", {git, "git://github.com/spawnfest/eneo4j.git", {branch, "master"}}}
]}.
